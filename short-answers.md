**SHORT ANSWER QUESTIONS**

**1. Difference between Google Knowledge Graph and Knowledge Panel**
The Knowledge Graph is Google’s database of entities and relationships, while the Knowledge Panel is the visible information box shown in search results.

**2. How Google determines entity identity**
Google determines entity identity using consistent signals like schema markup, brand mentions, backlinks, social profiles, and naming consistency across the web.

**3. When to create Custom Post Types instead of pages**
Custom Post Types are used when content is structured, repeatable, and is diffent from the blog post setup though and needs additional custom fields (e.g., courses, portfolios, testimonials, projects, properties etc).

**4. Recommended plugins for speed optimization and why**
LiteSpeed Cache, WP Rocket, Autoptimize, and Smush — they improve caching, reduce file size, and optimize images for faster performance.
